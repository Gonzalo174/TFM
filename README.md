Título del trabajo: Mapeo de fuerzas de tracción de células de epitelio mamario humano normal y tumoral
Directora: Dra. Lorena Sigaut
Codirectora: Dra. Lía Isabel Pietrasanta
Fecha de defensa: 26/03/2024
